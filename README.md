# steganography-project
A C-based steganography project that hides and extracts secret messages in images using the LSB technique, demonstrating file handling and bit-level manipulation.

# Steganography Project

## Overview
This project hides secret messages inside images using LSB technique.

## Features
- Encode message into image
- Decode hidden message

## Technologies
- C Programming
- File Handling

## How to Run
gcc encode.c decode.c -o stego
./stego
